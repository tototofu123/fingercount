A high-performance computer vision project that utilizes **MediaPipe** and **OpenCV** to recognize and count fingers in real-time. This project is part of the **Wood & Orange** series.

---

### 🚀 Quick Start
1. Install dependencies: `pip install opencv-python mediapipe`
2. Run the tracker: `python finger_counter.py`

---

### 🎨 Visual Identity
This repository follows the **Wood + Orange** theme. All UI overlays are rendered in `#F57C00` (Electric Orange) to ensure visibility and brand consistency.
